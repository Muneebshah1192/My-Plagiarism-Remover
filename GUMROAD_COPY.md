# Gumroad Product Copy

## Title
Originality Studio Pro — Netlify Rewriter & Writing Polish Tool

## Short description
A professional Netlify-ready web app for rewriting, polishing, and improving text with a no-key algorithm engine and optional AI Rewrite Assist.

## Full description
Originality Studio Pro is a clean, premium rewriting workspace built for creators, students, bloggers, and digital product sellers. It includes an Algorithm Rewriter that works without API keys and an optional AI Rewrite Assist mode that can be enabled with a server-side Gemini API key on Netlify.

The app helps users improve grammar, punctuation, clarity, sentence structure, and wording. It also includes similarity estimate, phrase overlap score, originality lift, readability score, TXT upload, TXT/HTML download, copy button, and local rewrite history.

## What buyers get
- Complete Netlify-ready source code
- Static frontend
- Netlify serverless function
- Algorithm rewriting engine
- Optional AI Rewrite Assist
- Premium responsive UI
- Deployment instructions
- Commercial-use starter template

## Honest disclaimer
This app rewrites and polishes wording. It does not check the internet and does not guarantee plagiarism removal. Users should cite borrowed ideas and review all output before publishing.
